Pastro.com - Cleaning Services Portal for Kosovo

A comprehensive web application that connects customers with professional cleaning companies across Kosovo cities.

## Features
- Company registration and management
- City-based directory
- Service search and filtering
- Financial dashboard for companies
- Multilingual support (Albanian/English)
- Payment integration
- Admin panel

## Tech Stack
- Next.js with TypeScript
- Prisma ORM
- TailwindCSS
- NextAuth.js
- PostgreSQL/SQLite

## Quick Start
1. Clone the repository
2. Install dependencies: npm install
3. Set up environment variables
4. Run database migrations
5. Start development server: npm run dev

## Demo Files
- demo-sweepsouth-style.html - Main homepage
- register-sq.html - Company registration
- signin-sq.html - User login
- Individual service pages for each cleaning service

## Database
Currently configured for SQLite (dev.db) for easy setup.
Can be switched to PostgreSQL for production.
